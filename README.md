# Interoperability

Transmute builds on open standards, and relies on them for interoperability.

## Identifiers

- [Decentralized Identifiers Spec](https://w3c.github.io/did-core/)
- [DID Spec Registries](https://w3c.github.io/did-spec-registries/)
- [Sidetree Spec](https://identity.foundation/sidetree/spec/)
- [DID Key](https://github.com/w3c-ccg/did-method-key)
- [DID Elem](https://github.com/decentralized-identity/element)

## Credentials

- [Verifiable Credentials Spec](https://w3c.github.io/vc-data-model/)
- [VC HTTP API](https://github.com/w3c-ccg/vc-http-api)

## Linked Data Signature Suites

- [Linked Data Proofs](https://w3c-ccg.github.io/ld-proofs/)
- [JSON Web Signature 2020](https://github.com/w3c-ccg/lds-jws2020)
- [Ed25519 Signature 2018](https://w3c-ccg.github.io/lds-ed25519-2018/)

## Services 

- [Encrypted Data Vault Spec](https://identity.foundation/secure-data-store/)

## Misc

- [JSON-LD](https://json-ld.org/spec/latest/json-ld/)
- [JOSE](https://www.iana.org/assignments/jose/jose.xhtml)
- [OpenID Connect](https://openid.net/specs/openid-connect-core-1_0.html)
